# gym-rtb

Install GYM enviroment
```bash
cd gym-rtb
pip3 install -e .
```

## Modules
```bash
pip3 install --upgrade https://storage.googleapis.com/tensorflow/mac/cpu/tensorflow-1.0.1-py2-none-any.whl
pip3 install keras
pip3 install matplotlib
```
